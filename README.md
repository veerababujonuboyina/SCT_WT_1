# Interactive Navigation Menu - SkillCraft Technology

This project is a modern responsive landing page developed as part of a web development task. The website features an interactive navigation menu that changes appearance on hover and scroll, ensuring an engaging user experience across all devices.

## Features

- Fixed navigation bar visible on all sections
- Hover animations and active menu highlighting
- Scroll-based navbar styling
- Smooth scrolling navigation
- Fully responsive design for mobile, tablet, and desktop
- Animated gradient background
- Glassmorphism UI effects
- Course showcase section:
  - Full Stack Development
  - Machine Learning
  - Data Science
- About SkillCraft Technology section
- Contact section with company information

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Objective

To create an interactive and responsive navigation menu that enhances user engagement through dynamic styling, smooth navigation, and modern UI design principles.
